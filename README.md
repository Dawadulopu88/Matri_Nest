# Matri_Nest
Overview
A full-stack web application designed to support expecting mothers throughout pregnancy and into the postpartum period. The platform centralizes health tracking, educational content, and curated resources in one accessible place.
Key Features

Health information tracking (symptoms, appointments, milestones through pregnancy stages)
Educational features covering trimester-specific guidance and postpartum care
Curated resource hub for maternal health support
Clean, responsive UI for ease of use during a stressful/important life stage

Tech Stack

Backend: Python, Django
Frontend: Bootstrap
Database: SQLite / MySQL

How It Works

Users register and create a profile tracking their pregnancy timeline.
Django models manage health records, appointments, and milestone data.
The system serves stage-specific educational content dynamically based on the user's current pregnancy week.
Bootstrap-based UI ensures the platform is usable across devices.
